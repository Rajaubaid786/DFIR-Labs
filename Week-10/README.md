## 📖 Overview

During Week 10 of the Cyberster DFIR Internship, I performed an in-depth Windows Registry forensic investigation using evidence extracted from a forensic disk image. The investigation focused on system configuration, user activity, USB device history, installed applications, browser artifacts, MRU records, and registry-based evidence to reconstruct user behavior and support a complete forensic investigation.

---

## 🎯 Objectives

- Analyze Windows Registry hives
- Investigate SYSTEM, SOFTWARE, SAM and NTUSER.DAT
- Examine USB device history
- Recover user account information
- Analyze MRU (Most Recently Used) artifacts
- Investigate installed applications
- Examine browser-related registry artifacts
- Correlate registry evidence with previous forensic findings
- Produce a complete forensic investigation report
