## 📖 Overview

During the final week of the Cyberster DFIR Internship, I completed a comprehensive insider threat investigation involving disk, memory, USB, registry, and network evidence. The investigation focused on identifying the suspect, reconstructing user activity, validating evidence integrity, and determining how confidential intellectual property was exfiltrated. Multiple forensic evidence sources were correlated to produce a complete investigative timeline and evidence-backed conclusions.

---

## 🎯 Objectives

- Perform forensic triage across multiple evidence sources
- Verify evidence integrity using cryptographic hashes
- Analyze Windows Registry artifacts
- Perform memory forensics using Volatility
- Investigate running processes and process hierarchy
- Recover command history and console activity
- Analyze USB device history
- Investigate network share (SMB) activity
- Examine browser history and user activity artifacts
- Correlate RAM, Disk, Registry and USB evidence
- Reconstruct the complete attack timeline
- Produce a professional DFIR investigation report
