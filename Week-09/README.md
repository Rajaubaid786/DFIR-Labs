## 📖 Overview

During Week 9 of the Cyberster DFIR Internship, I investigated application-layer artifacts by analyzing browser history, download records, SQLite databases, Windows Shell Link (LNK) files, and metadata. The investigation concluded with cross-artifact correlation to reconstruct user activity through a unified forensic timeline.

---

## 🎯 Objectives

- Perform browser forensic analysis
- Analyze SQLite browser databases
- Investigate download history and stored browser artifacts
- Examine Windows Shell Link (LNK) files
- Identify external device and volume activity
- Correlate browser, LNK, and previous forensic artifacts
- Build a unified forensic timeline
