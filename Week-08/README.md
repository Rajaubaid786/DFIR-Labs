## 📖 Overview

During Week 8 of the Cyberster DFIR Internship, I conducted a live Windows forensic investigation by analyzing authentication event logs, Windows Prefetch artifacts, browser cache, Thumbcache databases, and Recycle Bin metadata. The investigation focused on correlating multiple forensic artifacts to reconstruct user activity and build a defensible evidence timeline.

---

## 🎯 Objectives

- Analyze Windows Security Event Logs
- Establish user authentication baselines
- Investigate Windows Prefetch execution history
- Examine Browser Cache artifacts
- Analyze Thumbcache databases
- Investigate Recycle Bin metadata ($I / $R files)
- Correlate multiple forensic artifacts into a single investigation timeline
