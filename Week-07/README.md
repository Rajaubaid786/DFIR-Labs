## 📖 Overview

During Week 7 of the Cyberster DFIR Internship, I explored Windows file system forensics by analyzing NTFS artifacts, Alternate Data Streams (ADS), Master File Table (MFT) records, MACB timestamps, and conducting forensic case triage using Autopsy. The focus was on evidence acquisition, metadata analysis, and documenting forensic findings following industry best practices. :contentReference[oaicite:0]{index=0}
